REQUIRED ASSETS FOR THE APP
==============================

Please add the following image files to this directory:

1. farm-illustration.png (Main welcome screen illustration)
   - Dimensions: 1024x1024 or similar
   - Shows: A cow being milked with farm workers
   - Style: Cartoon/illustration style with warm colors
   - Format: PNG with transparency

2. icon.png (App icon)
   - Dimensions: 1024x1024
   - Format: PNG

3. adaptive-icon.png (Android adaptive icon)
   - Dimensions: 1024x1024
   - Format: PNG

4. splash.png (Splash screen)
   - Dimensions: 1242x2436 or larger
   - Format: PNG

5. favicon.png (Web favicon)
   - Dimensions: 48x48 or larger
   - Format: PNG

TEMPORARY SOLUTION:
-------------------
The app will display a placeholder view until farm-illustration.png is added.
You can export the illustration from the design file or create one to match the app theme.

TIPS:
-----
- Use high-resolution images for better quality
- Optimize images before adding to reduce app size
- Keep the farm theme consistent across all assets
